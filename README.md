## 🚀 My Own Project

### 📊 [Smart Resume Analyzer](https://github.com/sriram-projects/smart_resume_analyzer)
> Built from scratch — my own idea, design, and code

A Python-based CLI tool that helps job seekers understand how **Applicant Tracking Systems (ATS)** evaluate resumes.

**What it does:**
- 📄 Analyzes your DOCX resume against key industry keywords
- 📈 Gives a score out of 10 with actionable feedback
- 📝 Auto-generates a clean **PDF resume** and a personalized **cover letter**

**Tech:** `Python` `python-docx` `reportlab` `NLP` `ATS Logic`
