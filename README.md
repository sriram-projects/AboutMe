## 🚀 Featured Project

### 🤖 [Smart Resume Analyzer](https://github.com/sriram-projects/smart_resume_analyzer)
> *Built from scratch — my own idea, my own code*

An AI-powered resume tool that helps job seekers beat **ATS (Applicant Tracking Systems)** used by companies during recruitment.

**What it does:**
- ✅ Scores resumes against ATS keywords
- ✅ Gives personalized feedback to improve resume quality
- ✅ Auto-generates polished PDF resumes
- ✅ Creates matching cover letters automatically

**Why I built it:** I wanted to understand *how companies filter resumes using ATS* — so I built the tool myself to learn and solve that problem.

`Python` `NLP` `python-docx` `reportlab` `ATS` `AI`
